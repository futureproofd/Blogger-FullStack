# Blogger FullStack (MVC)  [![Build Status](https://travis-ci.com/futureproofd/Blogger-FullStack.svg?token=ckUzsBZ6S6JpCLYPq21N&branch=master)](https://travis-ci.com/futureproofd/Blogger-FullStack)

Taking an MVC Architecture-based approach (Model -> Service -> Controller(Routes) -> View(React)).

# Frontend
- ReactJS
- Redux

# Backend
- NodeJS
- Mongoose (MongoDB Atlas hosted)
- Redis Caching
- S3 Image hosting
- Passport Auth strategy (Google)

# Testing
- Jest
- Puppeteer
